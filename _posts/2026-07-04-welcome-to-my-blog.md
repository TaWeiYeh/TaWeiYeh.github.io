I've just rebuilt this site on Jekyll so I can finally post here without hand-writing HTML for every entry.

## How this works

Every post is a Markdown file in the `_posts/` folder, named `YYYY-MM-DD-title.md`, with a small front-matter block at the top:

```yaml
---
layout: post
title: "My next post"
---
```

Everything below the front matter is regular Markdown — headings, **bold**, lists, code blocks, and images all work as expected. Drop a new file in and it shows up on the [blog list](/blog.html) and homepage automatically, newest first.
